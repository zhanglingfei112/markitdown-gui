# Changelog

## v0.1.0 (2026-06-01)

### 🎉 Initial Release

- **Multi-format Support** — Convert PDF, DOCX, XLSX, PPTX, HTML, CSV, JSON, XML, EPUB, images, audio, and more to Markdown via Microsoft MarkItDown engine
- **Drag-and-Drop Interface** — Intuitive file import via drag-and-drop or file/folder selection dialogs
- **Batch Processing** — Queue hundreds of files for background conversion without UI freezing
- **Real-time Preview** — Dark-theme Markdown preview panel
- **Bilingual UI** — Full Chinese and English interface with instant switching
- **Flexible Export** — Export converted `.md` files to custom output directories with overwrite protection
- **Dark Theme** — Professional dark color scheme with consistent QSS styling
- **Configurable** — Customizable output directory, language preference, and optional LLM API key for enhanced OCR/image-to-text conversion
